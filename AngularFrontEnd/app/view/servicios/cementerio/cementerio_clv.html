TRÁMITE DE CONSERVACIÓN Y LIMPIEZA DE VÍAS DE CIRCULACIÓN DE NICHOS, SARCÓFAGOS Y URNAS
Inicio

¿Qué es el trámite de Conservación y Limpieza de Vías de circulación de Nichos, Sarcófagos y Urnas?

- Es el servicio para mantener las buenas condiciones y la limpieza de las vías donde se encuentran las sepulturas perpetuas, brindando así un aspecto agradable para todos los visitantes.

¿Qué necesito para realizar el pago en línea?

- Para encontrar al difunto debe conocer el código de ubicación del fallecido o los datos exactos del difunto (nombres, apellidos o CI).
- Para realizar el pago, previamente deberá tener habilitada su tarjeta de crédito o débito para realizar transacciones por internet.

¿Qué beneficios tengo al pagar mediante la plataforma web o móvil?

- Podrá pagar de manera fácil y rápida, evitando hacer filas y desde la comodidad de su hogar o donde se encuentre.
